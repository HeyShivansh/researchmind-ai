# ResearchMind AI

An enterprise-grade Retrieval-Augmented Generation (RAG) platform for scientific literature.

## Features

- Semantic Search
- Hybrid Retrieval
- Cross Encoder Reranking
- Citation-aware Answers
- Literature Review Generation
- Paper Comparison

## Tech Stack

Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- Qdrant

Frontend
- Next.js
- TailwindCSS

AI
- BAAI Embeddings
- Gemini
- Ollama
